v1.4.0 — 2026-04-21

- RSS Feed Attachments: attach one or more CSV or Excel (.xlsx/.xlsm) files to any job from
  the Notes window. Data is stored in the project JSON so it persists even if the original file
  moves. New "Attach RSS" button opens a file picker, shows a preview with a table-name field,
  and a Confirm button before saving. If feeds already exist, user is prompted to Replace All or
  Add New. The "RSS" button opens a feed viewer; if multiple feeds exist, a picker lets the user
  choose which to open.
- RSS viewer — admin features: sortable columns (click header to toggle asc/desc), Add Row
  (form dialog), double-click to edit a row, Delete Row (multi-select with confirmation),
  Delete RSS (removes feed from job with confirmation). All edits persist immediately.
- RSS viewer — non-admin feature: Propose Change button submits a description that is logged
  to the project's activity log for admin review.
- Bug fixes: resolved several PyCharm inspection warnings — QAbstractItemModel override
  signatures updated to accept QPersistentModelIndex, key_fn type annotation in list_projects,
  financials provider union type, unused variable, broad exception clause, and
  StandardButtons constructor usage.
- Performance: RSS table population skips resizeColumnsToContents for tables >500 rows to
  prevent UI freeze on large files; openpyxl active-sheet None guard added.

v1.3.0 — 2026-04-19

- Home Dashboard: landing screen shown when no project is selected; displays stat cards
  (Projects, Due This Week, Overdue, Total Tasks), top-5 projects by contract value,
  5 most recently added projects, and a live recent-activity feed (last 20 actions).
- Task Notes History: each task now has a timestamped note thread. Right-click any task
  and choose "Notes History" to view or add entries. Editing a task's Notes field also
  auto-appends the new text to the history thread. New TaskNoteRecord dataclass and
  backend methods (add_task_note, list_task_notes, delete_task_note, get_dashboard_stats).
- Keyboard shortcuts: Ctrl+N (new project), Ctrl+T (new task), Ctrl+F (project search),
  Ctrl+Shift+F (task search), Ctrl+E (export), Ctrl+Shift+E (bulk export), Escape (clear search).
- UI fix: scroll areas in the financials dialog and version history dialog now render with
  a transparent background, matching the dark navy theme correctly.
